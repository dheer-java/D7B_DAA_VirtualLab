# N-Queen Problem Virtual Lab

This project is a React-based Virtual Laboratory designed to simulate and visualize the **N-Queen Problem**, a classic algorithmic challenge solved using Backtracking.

## Overview
The N-Queen problem involves placing $N$ chess queens on an $N \times N$ chessboard so that no two queens threaten each other. This means no two queens can share the same row, column, or diagonal. 

This virtual lab provides an interactive way to explore this problem, serving as an educational tool to understand the underlying **Backtracking Algorithm**.

## Features
- **Interactive Simulator**: Visualize the placement of queens step-by-step on the chessboard.
- **Algorithm Execution Logs**: View real-time logs that trace the backtracking steps, providing a deeper understanding of the algorithm's decisions.
- **Theory Section**: A dedicated tab explaining the theoretical concepts, algorithms, and complexity analysis behind the N-Queen problem.
- **Modern UI**: Built with React and structured with Vite for a fast and smooth user experience.

## Tech Stack
- **Frontend Framework**: React
- **Build Tool**: Vite
- **Styling**: Vanilla CSS

## How to Run Locally

To run this virtual lab application on your local machine, follow these steps:

1. **Ensure Node.js is installed**: You will need Node.js and npm to run this project.
2. **Navigate to the project directory** (this directory).
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Start the Development Server**:
   ```bash
   npm run dev
   ```
5. **Open your browser** and navigate to the local URL provided in the terminal (typically `http://localhost:5173`). Have fun exploring the N-Queen problem!
